# Quintet

我在设计一款卡牌游戏。​把​一副扑克分为三个部分：​​
1. 牌堆：​​洗混的一副扑克，​​不含 Joker 共 52 张，​​暗牌。​​
2. 牌池：容量为 k（​​初始设定 1~5，开局后不作改变），从牌堆补充，明牌。
3. 牌阵：固定的​ ​5*5 共 25 个牌位组成的方形，​​明牌。​​

单人模式：​​
1. 开局设置牌池大小 k，​​从牌堆中抽取 k 张牌放在牌池。​​
2. 每个回合，玩家从牌池选择并放置一张牌到牌阵。除首张牌外，新放置的牌必须与至少一张已有牌在上下左右或对角方向相邻（共 8 个方向）。​​
3. ​当牌池清空且牌堆不为空时，​​从牌堆翻出补充至 k 张。​​
4. 玩家可选取任意位置作为起点，​​最终填满 5*5 区域即可。​​
5. 牌阵放满 25 张牌后，​​按每行、每列、两条对角线上的牌型计算总分（共 12 个牌型，​规则可参考德州扑克）。​​

双人模式：​​
1. 在单人模式基础上，​​双方共用一牌堆一牌池。每个回合轮流在牌池抽牌、分别构建自己的牌阵。​​
2. 双方构建的牌阵对对方明牌。​​且为了避免先手优势，​​可采用两局双方轮流先手、最终总分更高者取胜的方式。​​

分值计算参考（待根据概率调整）
- 皇家同花顺：50 + 最大牌点
- 同花顺：30 + 最大牌点
- 四条：24 + 四条牌点 * 2 + 踢脚牌点
- 葫芦：18 + 三条牌点 * 1.5 + 一对牌点
- 同花：14 + 五张牌点之和 * 0.2
- 顺子：12 + 最大牌点
- 三条：8 + 三条牌点 + 踢脚牌点
- 两对：5 + 高对牌点 + 低对牌点 * 0.5 + 踢脚牌点
- 一对：2 + 对子牌点
- 高牌：0 + 最高牌点
